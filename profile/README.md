---

agenthatch turns a SKILL.md into a working agent. Tool calling, multi-turn
conversation, MCP — all wired up and ready to run. No prompt injection hacks,
no YAML-driven state machines. Just a Python package you can import and ship.

The long bet: a single general-purpose Agent that spawns specialized Agents
whenever you need one. You won't build pipelines. You won't hand-craft domains.
One hatch, infinite agents.
